## 📝 Resumo

O projeto tem como objetivo o desenvolvimento de um aplicativo mobile voltado para o gerenciamento de partidas amistosas de futebol na região, permitindo que usuários organizem jogos, registrem resultados e interajam entre si por meio de funcionalidades semelhantes a uma rede social, como curtidas e compartilhamentos.

A proposta surge da dificuldade encontrada por jogadores amadores e times locais em organizar partidas e manter registros confiáveis dos resultados. A pesquisa é baseada na análise de aplicativos esportivos e redes sociais já existentes, identificando limitações quanto à organização de jogos informais.

Como resultado esperado, pretende-se criar uma aplicação funcional, intuitiva e acessível, que facilite a comunicação entre jogadores, promova a integração da comunidade esportiva local e contribua para a organização das atividades esportivas, utilizando tecnologias modernas de desenvolvimento mobile e banco de dados.

# 📌 Plano de Pesquisa

## 📖 Introdução

O futebol é uma das práticas esportivas mais populares no Brasil, sendo amplamente praticado de forma amadora em bairros, escolas e comunidades. No entanto, a organização de partidas amistosas ainda enfrenta diversos desafios, como a dificuldade de reunir jogadores, registrar resultados e manter um histórico confiável das partidas realizadas.

Muitas vezes, essas informações se perdem ou ficam restritas a grupos informais em aplicativos de mensagens, o que dificulta a gestão e o acompanhamento das atividades.

Além disso, não há uma ferramenta específica voltada para esse público que una organização de partidas com interação social. Aplicativos existentes geralmente focam em esportes profissionais ou não oferecem recursos de engajamento entre usuários, como publicação de resultados, curtidas e compartilhamentos.

Diante desse cenário, surge a necessidade de desenvolver uma solução tecnológica que atenda às demandas dos jogadores amadores, facilitando a organização de partidas e promovendo a interação entre os participantes. O desenvolvimento de um aplicativo mobile com essas funcionalidades pode contribuir significativamente para a melhoria da experiência dos usuários, incentivando a prática esportiva e fortalecendo a comunidade local.

## 🎯 Objetivos

### Objetivo Geral

Desenvolver um aplicativo mobile para gerenciar partidas amistosas de futebol, permitindo a organização de jogos, registro de resultados e interação social entre usuários.

### Objetivos Específicos

* Levantar os requisitos do sistema com base nas necessidades dos usuários
* Modelar o banco de dados da aplicação
* Desenvolver a interface mobile de forma intuitiva e responsiva
* Implementar funcionalidades de criação e gerenciamento de partidas
* Desenvolver sistema de publicação de resultados com curtidas e compartilhamentos
* Realizar testes para validação do sistema

---

## 🛠 Metodologia

O presente projeto caracteriza-se como uma pesquisa aplicada, pois tem como objetivo desenvolver uma solução prática para um problema real enfrentado por jogadores de futebol amador. Também possui caráter exploratório, uma vez que busca compreender melhor as necessidades dos usuários em relação à organização de partidas esportivas.

Além disso, será realizada uma pesquisa bibliográfica, com base em artigos, documentações e materiais relacionados ao desenvolvimento de aplicativos mobile, banco de dados e usabilidade.

O desenvolvimento do projeto seguirá as seguintes etapas:

* Levantamento de requisitos
* Modelagem do sistema
* Desenvolvimento da interface
* Implementação das funcionalidades
* Integração com banco de dados
* Realização de testes

As ferramentas utilizadas poderão incluir:

* Android Studio
* Java ou Kotlin
* Firebase ou MySQL

A coleta de dados poderá ser feita por meio de observação e análise do comportamento de usuários em contextos semelhantes.

## 📊 Resultados Esperados

Espera-se como resultado a entrega de um aplicativo mobile funcional e intuitivo, capaz de auxiliar na organização de partidas amistosas de futebol.

O sistema permitirá:

* Cadastro de usuários
* Criação de partidas
* Registro de resultados
* Interação social entre participantes

O projeto busca beneficiar jogadores amadores e comunidades locais, facilitando a comunicação, organização e engajamento entre os usuários. Além disso, pretende-se reduzir a desorganização e a perda de informações, promovendo uma melhor experiência na prática esportiva.
